Hello World!

"Hello World!" is a very popular phrase in computer programming. It was first used in the 70's by the authors of a C language book. Awesome!


Hello World! as a program

As a program, it's a simple implementation that displays this text, with the goal of portraying it into the screen. It's mainly used for beginners in a language and to test if a programming environment is operational.



Importance of "Hello World!"

  • It's a standard in the industry.
  • It's one of the shortest and easiest ways of showing a programming language.
  • It's a sign of developer pride.
